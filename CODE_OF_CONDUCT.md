# Code of Conduct

## Our standard

This is a personal technical project maintained by **crisis1er**.  
All interactions must remain respectful, constructive, and focused
on the technical subject.

## Expected behavior

- Use clear, respectful language
- Stay on topic — Squid, openSUSE, proxy, security, privacy
- Accept and give constructive feedback gracefully
- Credit others' work when building upon it

## Unacceptable behavior

- Harassment, insults, or discriminatory language of any kind
- Spam or self-promotion unrelated to the project
- Publishing others' private information without consent
- Any behavior that would be inappropriate in a professional setting

## Enforcement

Issues or PRs that violate this code of conduct will be closed
without further discussion. Repeat violations will result in blocking.

## Scope

This code of conduct applies to all spaces managed by crisis1er:
GitHub repositories, issues, pull requests, and discussions.
